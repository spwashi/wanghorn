ANCHOR

AMPERSAND
    represents where an anchor might belong, but there is not an anchor


    - Anchor ID     (or null)
    - Ampersand ID  (not null)


APPLICATION
    represents the combination of an Integration Scheme (#)
                               or an Anchor             (&)
